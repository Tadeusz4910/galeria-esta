## 1. GŁÓWNA IDEA PROJEKTU

Nowa strona Galerii ESTA nie jest zwykłą modernizacją obecnej strony
ani tylko nowym katalogiem prac. Jest nowoczesną platformą prezentacji,
wiedzy, sprzedaży i kolekcjonowania polskiej sztuki powojennej
i współczesnej, opartą na realnym dorobku Galerii ESTA z 28 lat działalności.

**Najważniejsze założenie:**
Galeria ESTA nie zaczyna od zera. Nowa strona wykorzystuje dotychczasowy
dorobek, zasób prac, archiwum, wystawy, targi, relacje z artystami
i doświadczenie rynku — przedstawiając je w nowej, uporządkowanej,
kuratorskiej i sprzedażowej formie.

Nowa strona jest jednocześnie:
1. Wizytówką Galerii ESTA — 28 lat działalności, doświadczenie, pozycja
2. Platformą wiedzy — tłumaczącą sztukę konceptualną, konkretną, geometryczną
3. Publiczną kolekcją online — wybrane prace dostępne publicznie
4. Systemem viewing-room — oferty prywatne, kuratorskie, sprzedażowe
5. Centrum kompendium — porządkującym sztukę przez pojęcia, nie nazwiska
6. Miejscem prezentacji wystaw i targów — jako dowód działalności
7. Bramą do archiwum — stara strona pozostaje jako historyczny zasób
8. Narzędziem sprzedażowym — wspierającym stabilność finansową
9. Platformą międzynarodową — dla odbiorców zagranicznych

## 2. POZYCJONOWANIE

**Galeria ESTA — polska sztuka powojenna i współczesna
przez idee, które ją kształtują.**

**Hasło główne:**
Idea. Słowo. Obraz. Struktura. Światło. Pamięć. Proces.

**Wersja angielska:**
Idea. Word. Image. Structure. Light. Memory. Process.

**Główna myśl strategiczna:**
Galeria ESTA prezentuje polską sztukę powojenną i współczesną poprzez idee,
które ją kształtują: słowo, znak, obraz, geometrię, światło, przestrzeń,
pamięć i proces.

**Ważna korekta strategiczna:**
Nie piszemy: "Galeria ESTA jest tylko galerią sztuki konceptualnej."
Piszemy: "Galeria ESTA buduje nowoczesną platformę wokół polskiej sztuki
powojennej i współczesnej, której główną osią interpretacyjną są:
idea, słowo, obraz, struktura, światło, pamięć i proces."

## 5. ARTYŚCI PROGRAMU

### Rdzeń konceptualno-konkretny

**Jarosław Kozłowski** — idea / język / logika / paradoks / system
Jedna z kluczowych postaci polskiego konceptualizmu.
Twórczość skupiona na języku, pojęciu, definicji, paradoksie.
Dzieło jako sytuacja intelektualna.

**Stanisław Dróżdż** — słowo / znak / poezja konkretna / przestrzeń języka
Oś sekcji dotyczącej poezji konkretnej.
Słowo jako obraz, znak, układ przestrzenny, struktura znaczeń.

**Wanda Gołkowska** — struktury otwarte / system / geometria / proces / odbiorca
Łączy geometrię, sztukę konkretną, struktury otwarte
i konceptualne myślenie o dziele jako układzie możliwości.

**Barbara Kozłowska** — przestrzeń / działanie / proces / zapis / efemeryczność / ślad
Poszerza program o sztukę działania, przestrzeni, czasu, zapisu.

**Jan Chwałczyk** — światło / cień / projekcja / zjawisko / relacja obiektu i przestrzeni
Kluczowy dla sekcji światła i cienia.
Prace badające relację między materialnym obiektem a niematerialnym zjawiskiem.

**Zbigniew Gostomski** — geometria / struktura / rytm / przestrzeń / system wizualny
Mocny filar sekcji geometrii, struktury i systemu.

**Andrzej Dłużniewski** — język / absurd / filozofia / poetyka pojęcia
Ton poetycki, absurdalny, filozoficzny i egzystencjalny.

**Natalia Brandt** — współczesna kontynuacja / geometria / konkret / struktura
Pokazuje że język konkretu i geometrii nie jest tylko historyczny.

### Program poszerzony

**Marek Sobczyk** — obraz / tekst / ironia / Gruppa / malarstwo krytyczne
Obraz krytyczny, tekst w obrazie, ironia, malarstwo po konceptualizmie.

**Jerzy Lewczyński** — fotografia / pamięć / archiwum / ślad / archeologia obrazu
Fotografia jako pamięć, ślad, archiwum, dokument.

**Andrzej Paruzel** — wideo / instalacja / media / proces / percepcja
Media eksperymentalne, wideo, instalacja, proces, percepcja.

**Grupa Twożywo** — słowo / znak / typografia / miasto / komunikat / przestrzeń publiczna
Poezja konkretna, typografia, komunikat społeczny, plakat, mural.

**Mieczysław Wiśniewski** — geometria / układ / relief / światło / struktura / powierzchnia
Geometria, układ, struktura, światło i reliefowe myślenie o obrazie.
